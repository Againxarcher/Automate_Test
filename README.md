## Requirements
* Python 3.x
* Robot Framework
* Robot Framework Seleniumlibrary
* Browser driver (ChromeDriver)

You may install the prerequisites by running:

    pip3 install -r requirements.txt

## Execute basic setup test run:

    $ robot tests/web_page_data_check.robot
